Fite Time

Fite Time is an iOS application for Mixed Martial Arts (MMA). It provides upcoming event schedules, fight cards details, and live results update. In an upcoming version, user will also be able to score the fights as it unfolds.

For this application, I built the entire backend plus client. Backend was built on AWS products, such as DynamoDB and Lambda functions. The newer portion of the app uses Google cloud products. For example, user score section involves both the Realtime and Firestore database. Cloud functions are used to manage background work.

This application uses both Objective-C and Swift. This project uses core data, notifications, and custom animations.

Learn Basic Japanese

This untitled application was born from my desire to learn Japanese and my frustration with Duolingo. I was an avid user of Duolingo, which tries to turn learning a new language into a game. However in the end I find it to be an ineffective way of learning Japanese. I built an app that helps me learn Japanese, the hard way, and it may get released into the app store in the future.

The application is essentially powered up flash card. Tapping on the card plays the sound of the Hiragana or Katakana character. There's also a section on practicing writing the characters. This is important because writing helps to reinforce learning new characters. Please note the stroke order on the characters. I spent a lot of time making those files and I think they turned out well!

Progress Bar Animation

This project demonstrates how to animate a progress bar. The animation here is set to decrement from 10 seconds to 0 seconds. Animation is performed every second over 1 second.
